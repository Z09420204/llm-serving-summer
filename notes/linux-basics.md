# Linux Basics

Date: 2026-07-22

## Paths

- `/`：文件系统根目录
- `~`：当前用户主目录
- `.`：当前目录
- `..`：上一级目录
- Absolute path：从 `/` 开始的完整路径
- Relative path：从当前目录开始的路径

## Commands

| Command | Meaning |
| --- | --- |
| `pwd` | 显示当前目录 |
| `ls` | 查看目录内容 |
| `ls -la` | 查看详细信息和隐藏文件 |
| `cd` | 切换目录 |
| `mkdir` | 创建目录 |
| `touch` | 创建空文件 |
| `cat` | 显示文件内容 |
| `cp` | 复制文件 |
| `mv` | 移动或重命名文件 |
| `rm` | 删除文件 |
| `rmdir` | 删除空目录 |
